# prompt-website

A full-stack web application built with [Maestru](https://maestru.com).

## Stack

- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Express + TypeScript
- **Database:** PostgreSQL + Drizzle ORM
- **Deployment:** Railway

## Getting Started

```bash
npm install     # Install dependencies
npm run dev     # Start dev servers
```

Frontend runs on [localhost:3000](http://localhost:3000), API on [localhost:3001](http://localhost:3001).

## Project Structure

```
apps/web/       React frontend
apps/api/       Express backend
packages/       Shared code
.maestru/       Documentation and specs
```

## Documentation

Project documentation lives in `.maestru/docs/`. Use `maestru search <query>` to find anything.
